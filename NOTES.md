Kap (Mac) - clean, simple

Records screen to GIF or MP4
Good compression
https://getkap.co/
Open Kap
Drag the recording frame over your app
Hit record
Do your interaction (filter the table, answer a question)
Stop recording
Trim if needed
Export as GIF (or MP4 - think about this in lieu of gif)
Frame rate: 10-15 fps (smaller files)
Size: 1000px width max
Format: Try MP4 first, then GIF if you prefer




Giphy Capture (Mac) - dead simple

Record → trim → export
https://giphy.com/apps/giphycapture

Record at 10-15 fps (not 30) - smaller file size
Keep GIFs under 2-3 MB each
5-10 seconds max
ONE key interaction per GIF
- Greyhound table: Filter dropdown → results updating
- Der Die Das: Answer a question → beer stein filling
